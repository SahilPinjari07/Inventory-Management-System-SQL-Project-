# 🗄️ Inventory Management System (SQL Project)

## 🚀 Overview
This project is a **SQL-based Inventory Management System** designed to manage and analyze vehicle sales, spare parts inventory, customer data, and supplier records. It enables efficient tracking of inventory, monitoring of sales performance, and generation of actionable business insights.

---

## 🎯 Objectives
- Track vehicle sales and revenue trends  
- Monitor inventory levels and generate low-stock alerts  
- Manage customer and supplier data  
- Automate inventory updates and transaction processes  

---

## 🛠️ Tools & Technologies
- **MySQL**  
- **SQL (Joins, Aggregations, Subqueries)**  
- **Database Design & Normalization**

---

## 📌 Key Features
- 📊 Sales analysis (monthly & yearly trends)  
- 🚗 Identification of top-selling vehicles  
- 📦 Inventory tracking with low-stock alerts  
- 🧾 Automated invoice generation using SQL triggers  
- 🔄 Real-time inventory updates after sales transactions  

---

## 🗂️ Database Structure
The system is built using multiple relational tables:
- **Customers** – Stores customer details  
- **Vehicles** – Contains vehicle information  
- **Sales** – Records transaction data  
- **Suppliers** – Maintains supplier details  
- **Inventory** – Tracks stock levels
